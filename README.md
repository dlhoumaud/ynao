# Ynao

![Drapeau](./flag.png)

**Ynao** (la communication avec l'autre pour agir sur le monde) est un framework linguistique et une langue hybride humain–machine conçue pour :

* communication humaine naturelle et incarnée
* interprétation machine stable
* génération lexicale structurée
* évolutivité contrôlée et description physique de la matière (agglutination)

## Le drapeau et la philosophie

La devise gravée sur le blason guide l'esprit du projet :
> Avoir la foi, c’est accepter l’absurde qui se cache derrière la montagne.
> Refuser l’absurde, c’est choisir de mourir immobile.

Ce drapeau est celui des esprits qui n'ont pas peur de l'inconnu. L'oiseau noir (le corbeau, symbole d'intelligence adaptative et associative) survole la montagne de l'arbitraire. Il tient au centre le noyau géométrique de l'Ynao : l'outil logique qui permet de traverser le chaos pour lui donner une structure et du sens.
