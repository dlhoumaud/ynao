# Ynao — Documentation Canonique

### Langue humain–machine générative polysynthétique

**Statut :** prototype structuré cohérent (Intégration de la matière, de la géométrie et de l'espace)

---

# 1. Vision générale

**Ynao** (la communication avec l'autre pour agir sur le monde) est une langue hybride humain–machine conçue pour :

* communication humaine naturelle et incarnée
* interprétation machine stable
* génération lexicale structurée
* évolutivité contrôlée et description physique de la matière (agglutination)

---

# 2. PHONOLOGIE ET GÉOMÉTRIE

Ynao privilégie la stabilité computationnelle sur l’intuition phonétique native.
Elle est pensée pour être parlée et manipulée par les humains et les machines sans friction.
La difficulté de prononciation pour des apprenants est un coût d’apprentissage assumé et commune à toutes langues.

Chaque graphème doit :
- exister sur les claviers internationaux standards
- être facilement sérialisable et parsable
- posséder une correspondance phonétique stable et unique
- éviter les diacritiques et caractères locaux
- rester compatible avec les systèmes numériques mondiaux

# 3. STRUCTURE GÉNÉRALE

## 3.1 Conjugaison

Absente. Les verbes sont invariants.

## 3.2 ÊTRE

Implicite. L’absence de verbe indique un état ou une identité.

* `me David` → je suis David
* `nu Vupi` → tu es Vupi
* `me gera` → je suis calme

## 3.3 Singulier & Genre

Singulier : Implicite.
Genre (pronoms) : Optionnel.
Les noms communs sont strictement neutres/sans genre grammatical.

## 3.4 Sémantique

Les mots Ynao sont utilisables directement, décomposables phonologiquement et reconstructibles structurellement.

# 17. PHILOSOPHIE DU LANGAGE

Ynao repose sur :

* phonologie générative et syntaxe minimale
* polysynthèse conceptuelle (le mot est la définition géométrique et fonctionnelle de l'objet)
* lisibilité machine et ergonomie humaine

# 18. FORMATION AGGLUTINATIVE DES OBJETS (LEXIQUE PHYSIQUE)

Les objets matériels n'ont pas de mots arbitraires. Ils sont définis par leur fonction, leur géométrie et leur interaction physique avec le monde.

Ynao applique une règle de **compression sémantique** :

> plus un mot devient un objet lexical stable, plus sa structure interne est condensée.

Les **prépositions syntaxiques** (`ni`, `se`, `so`, `sofo`, `sebo`) restent pleinement utilisées dans les phrases, mais sont généralement **supprimées dans les mots composés**, sauf lorsqu’elles sont nécessaires pour préserver le sens spatial fondamental de l’objet.

## 18.1 Règles de construction

### 1. Base fonctionnelle

Le mot commence généralement par un **verbe racine** représentant la fonction dominante.

| Fonction              | Racine |
| --------------------- | ------ |
| bouger / déplacement  | `vo`   |
| savoir / connaissance | `sro`  |
| créer / fabriquer     | `po`   |

---

### 2. Compression structurelle

Les éléments géométriques ou fonctionnels sont ajoutés dans leur forme minimale.

**Voyelles-formes :**

| Voyelle | Forme                      |
| ------- | -------------------------- |
| `a` `A` | triangle                   |
| `e` `E` | rectangle / boîte          |
| `i` `I` | tige / ligne               |
| `o`     | rond                       |
| `u`     | cuve / réceptacle          |
| `y`     | intersection               |
| `x`     | croisement                 |
| `k`     | angle                      |
| `l` `L` | angle droit                |
| `c`     | demi cercle ou croissant   |

---

### 3. Prépositions implicites

Dans les mots composés :

* `ni` (avec) est généralement implicite
* `se` (dans) est supprimé sauf s’il porte le sens principal
* `so`, `sofo`, `sebo` sont conservés uniquement si la position spatiale définit l’objet

---

### 4. Pluriel interne inversé

Quand un élément interne est pluriel, le suffixe est inversé :

`oc` → `co`

Cela permet de garder le pluriel final disponible.

Exemple :

roues (`oc`) → `co`

## 18.2 Exemples canoniques

| Objet   | Mot Ynao | Construction                                                   |
| ------- | -------- | -------------------------------------------------------------- |
| voiture | `voeco`  | bouger (`vo`) + rectangle (`e`) + roues (`co`)                 |
| avion   | `vohca`  | bouger (`vo`) + air/abstraction (`h`) + ailes/triangles (`ca`) |
| vélo    | `voico`  | bouger (`vo`) + tige (`i`) + roues (`co`)                      |

## 18.3 Principe général

Un mot Ynao doit tendre vers :

> **la forme minimale conservant le maximum de sens**

Si une partie du mot n'apporte pas d'information distinctive, elle peut être supprimée.

## 18.4 hiérarchie lexicale

Dans une logique de fréquence lexicale, les mots les plus communs ont une compression plus forte.

Lorsqu’un concept plus rare partage une structure proche d’un concept central, c’est le concept rare qui reçoit les marqueurs phonologiques additionnels.

> Si un mot proche de *voiture*, comme *karting* par exemple. Alors c’est le mot *karting* qui doit porter les marqueurs phonologique additionnels.

Règle de dérivation lexicale Ynao :

1. Identifier le prototype ontologique.
2. Choisir la forme minimale du prototype.
3. Ajouter uniquement les traits distinctifs.
4. Ordonner les traits :
   fonction > matière > géométrie > comportement > contexte
5. Compression si collision nulle.

Exemples arbitraires :

véhicule terrestre à roues
└── voiture = prototype central `voeco`
    ├── karting `vobecox` (bouger `vo` + petit `b` + cube `e` + avec des roues `co` + energique `x`)
    ├── camion `vofeco` (bouger `vo` + grand/gros `f` + cube `e` + avec des roues `co`)
    ├── taxi `voecoaro` (bouger `vo` + cube `e` + avec des roues `co` + dont l'action `a` + est en relation `r` + avec les autres `o`)
    └── voiture de course `voecox` (bouger `vo` + cube `e` + avec des roues `co` + energique `x`)

## 18.5 Règles de liaison phonologique (Nouvelle Mouture)

Ynao applique le Principe de Compacité Maximale. Les voyelles de liaison ne sont pas là pour décorer ou systématiser, mais uniquement pour résoudre un blocage phonatoire réel (impossibilité physique de prononcer la suite de consonnes à l'oral).

### 1. Règle de la Césure Syllabique (Par défaut)

Toute succession de deux consonnes est autorisée sans voyelle de liaison si elle s'articule via une frontière de syllabe naturelle (VC−CV). La première consonne ferme la syllabe précédente (coda), la seconde ouvre la syllabe suivante (attaque).

> **Exemples fluides directs :** uimvo (uim-vo), uitvo (uit-vo), uipvo (uip-vo), uibco (uib-cho).  

### 2. Déclencheurs de la Voyelle de Liaison (Les Exceptions)

Une voyelle de liaison (e, a, i) est insérée uniquement et strictement dans les deux cas de blocage suivants :

> **Collision Homorganique (Identique) :** Deux consonnes strictement identiques ou trop proches phonétiquement qui se suivent (ex: tt, kk, dd). Raison : l'oreille humaine n'entendrait qu'une seule consonne longue.

> **Collision Tautosyllabique (Inarticulable) :** Un groupe de consonnes qui ne peut pas se séparer en deux syllabes et qui force un blocage de la glotte ou de la langue (ex: tg, qdb, xcd).

> **Règle d'or :** Si un humain peut prononcer l'agglutination de manière fluide à vitesse normale, le mot reste compact. Pas de voyelle de liaison artificielle.

---

### 3. Voyelles de liaison

Lorsqu’une collision phonologique apparaît, une voyelle de liaison est automatiquement insérée selon sa position dans le mot.

#### Début du mot

Ordre prioritaire :

`e → a → i`

Si plusieurs voyelles de liaison successives sont nécessaires, elles ne peuvent pas être identiques.

Exemple :

`ee` → interdit
`ea` → autorisé

#### Milieu du mot

Ordre prioritaire :

`a → i → e`

Exemple :

`xcd` → `fudoxacid`

#### Fin du mot

Ordre prioritaire :

`i → a → e`

**Exception canonique :**

La dernière lettre d’un mot ne doit jamais être une voyelle de liaison.

Si une liaison serait nécessaire uniquement pour terminer le mot, elle est supprimée.

Exemple :

`pefadi` → non recommandé
`pefad` → forme canonique

---

### 4. Déterminisme

Pour une même structure profonde, une seule forme phonologique valide doit exister.

Une entrée identique doit toujours produire exactement le même mot prononçable.

Cette règle garantit :

* stabilité machine
* reconstruction lexicale
* absence d’ambiguïté
* cohérence inter-locuteurs

La phonotactique Ynao est donc **déterministe**.
